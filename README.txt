Tarea 4 Matematicas Computacionales
Escribir parser (Bison) y scanner (construir con flex) para convertir archivo de XML producido por JFLAP a 
definicion a quintuplo (K, Sig, F, s,f) donde:
K = Conjunto de estados del automata.
Sig = Alfabeto del automata
F = Funcion de transicion
s = Estado inicial
f = Conjunto de estados finales

Ejemplo de entrada y salida en PDF: Tarea4.pdf

El programa debe de incluir comentarios para cualquier fragmento de codigo,
estructuras, variables reglas y funciones que no sean explicitamente evidentes.

Los datos se leen por entrada estandar.
La salida de resultados debe ser por salida estandar.
Debe ser posible redirigar la entrada y salida usando pipas e.g. <, |, >.

Solo una persona subira la tarea en un archivo zip M_Tarea4.zip donde M son las matriculas del equipo.
El zip contiene jflap2afd.l y jflap2afd.y
Cada archivo fuente debe tener las matriculas y nombres
La asignacion de blackboard tambien debe incluir las matriculas
